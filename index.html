Client

You can download the source code for the client here: https://github.com/MTUCappuccino/Helium

To build, open a command line and type ant. You should be able to launch the executable jar like any other java application.

If the server is running, enter the address and port of the server, as well as a handle and a password (if required). Press connect. You should now be able to send messages and receive messages.

Server

You can download the source code for the server here: https://github.com/MTUCappuccino/Helium_Server

To build, open a command line and type ant. You need to launch the server from the command line by using java -jar Helium_Server.jar.

Commands:

make - starts a new server. The server will ask the following questions:

  • Port to open
  • Server Name
  • Password entry?(Y/N)
  • Password (if you answered yes above)

There is currently no way to gracefully close the server. Use Ctrl+C to force quit.

Known issues:

  • The server crashes if a client disconnects
  • There is no error handling if the user inputs invalid arguments
